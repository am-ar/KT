Static website

A static website is a type of website that delivers the same content to every user, regardless of who they are or how they access the site. The content is typically created using HTML, CSS, and sometimes JavaScript, and is stored on a server without a database or server-side scripting.

Key characteristics of static websites include: Fixed Content: Each page is a fixed file; changes require manual updates to the files.
Speed: Static sites generally load faster since they don’t require database queries or server-side processing.
Simplicity: They’re simpler to host and can be served from any basic web server or even a content delivery network (CDN).
Security: Fewer vulnerabilities compared to dynamic sites since there are no databases or server-side languages involved.
